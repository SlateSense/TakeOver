                    HOW TO CHECK IF MINER IS WORKING

📁 Double-click: CHECK_STATUS.bat

Shows:
✅ Is miner running?
✅ Current hashrate
✅ Pool connection status
✅ Shares accepted/rejected
✅ Uptime

Takes: 2 seconds

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

METHOD 2: CONTINUOUS MONITORING

📁 Double-click: MONITOR_CONTINUOUS.bat

Shows:
✅ Live hashrate updates every 10 seconds
✅ Real-time monitoring
✅ Press Ctrl+C to stop

Use this to:
- Monitor during competition
- Watch hashrate in real-time
- Verify miner is stable

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

METHOD 3: DETAILED CHECK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Shows everything:
✅ CPU information
✅ Huge pages status
✅ File locations
✅ Network connections
✅ Full statistics

TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Miner NOT RUNNING"
   → Run DEPLOY_ULTIMATE.ps1 again

⚠️  "Low CPU usage"
   → Miner may be stuck
   → Wait 15 seconds (watchdog will restart it)
   → Or manually restart with DEPLOY_ULTIMATE.ps1

⚠️  "API NOT ACCESSIBLE"
   → Normal if API disabled for stealth
   → Check process status instead
   → If process running + high CPU = working fine

⚠️  "Huge Pages 0%"
   → Hashrate will be 20% lower
   → Huge pages privilege may not be granted
   → Redeploy to fix
