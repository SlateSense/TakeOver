================================================================
 🔧 TROUBLESHOOTING
================================================================

ERROR: "File not found"
→ Make sure you're in the correct folder
→ All files must be together

ERROR: "Access denied"
→ Run as administrator
→ Or use RUN_ENHANCED.bat instead

ERROR: "Defender blocked"
→ Windows Security → Allow the file
→ Or temporarily disable real-time protection

ERROR: "Script won't start"
→ Check xmrig.exe is present
→ Try CHECK_STATUS.bat first
